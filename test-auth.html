Test Google Authentication

Open the browser console and check the logs when you test Google authentication.

Go to http://localhost:3000/login to test.

Debugging Steps:

  1. Open browser console (F12)
  2. Go to login page
  3. Click "Continue with Google"
  4. Complete Google authentication
  5. Check console logs for:
    • "FirebaseAuthContext: Dispatching auth-token-updated event"
    • "AuthContext: Received auth-token-updated event"
    • "AuthContext: Loading user profile..."
    • "AuthContext: Token found: true"
    • "AuthContext: Making API call to /api/auth/profile"
    • "AuthContext: Profile response: ..."

Common Issues:

  • If you see "Token found: false" - Firebase authentication didn't work
  • If you see "Error fetching user profile" - Backend API issue
  • If you don't see the auth-token-updated event - Context communication issue
